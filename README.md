### wx-translate - 小程序翻译应用 - 2019.02

<div align="left">
<img src="https://i.loli.net/2019/02/27/5c7656307e497.jpeg" width="400">
</div>

- <u>[wx-translate](https://github.com/strugglebak/wx-translate)</u> 这是一个基于微信小程序平台的翻译小工具,支持英、中、日、韩、法、西班牙、阿拉伯语言的在线翻译,支持语言类型的自动识别,支持历史记录的保留和查看,支持多行输入翻译等功能,后端 API 主要使用百度翻译,翻译失败提示采用小程序自带 API 实现.

技术栈: 页面注册 + 模块化 + 数据响应和绑定 + flex 布局 + Promise 异步控制
